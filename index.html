Okay so I am learning Scala. I have background in web apps, so I thought best way to learn will be build web app using Play Framework.

But what type of app to build. Todo apps are boring though they lets you grasp basic pretty fast.

What should I build.

I think a Twitter like social network would be great way to learn about Scala. Also twitter also uses Scala in backend.

Getting started page of play-framework says bootstrapping hello world project is very easy.

Create a directory.

$ mkdir me-social
$ cd me-social

Run following command to create a play project.

$ sbt new playframework/play-scala-seed.g8
